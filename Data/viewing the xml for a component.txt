you can view the xml for a component by clicking the view component xml icon for a specific revision in the revision history dialog. procedure in the component explorer, navigate to the folder containing the component. double-click the component. the component opens on a tab. click revision history. the revision history dialog opens showing each revision. click view component xml next to a revision number. the component xml window opens showing the xml that makes up the selected revision of the component. click close to close the component xml window.
