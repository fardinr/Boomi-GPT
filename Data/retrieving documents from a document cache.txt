use a retrieve from cache shape to retrieve documents from a document cache so that they can be used in the currently executing process or subprocess. procedure drag the retrieve from cache shape onto the process canvas. the retrieve from cache shape dialog opens. optional: enter a display name for the shape. select or create a document cache component. indicate which documents should be retrieved from the specified document cache: all documents — retrieves all documents from the document cache. indexname — retrieves documents that are associated with the selected index. note: retrieving all documents from the document cache is available only if the enforce one index entry per document option is in effect for the specified document cache component. if that option is not in effect, documents can be retrieved only by index name. optional: if you are retrieving documents by index name, specify a parameter value for each key within the selected index. click ok.
