you can set a property value in the beginning of the process and then retrieve it later in a different part of the process. a dynamic process property is a single "on-the-fly" property that you create. procedure open a process and add or configure a step (such as the set properties step). click the add property icon. the choose property dialog opens. select dynamic process property. type a property name. optional: to persist the property value across subsequent executions, turn on persist this property. click ok. click ok to save the step. click save to save the process.
