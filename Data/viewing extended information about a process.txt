you can view extended process information in process reporting. procedure select manage > process reporting. find the desired process execution result. do one of the following: in the actions menu for that execution, select view extended information. click the date/time link for that execution to go to its detail view and in the actions menu, select view extended information. the extended execution information dialog opens. view the ids displayed in the dialog. optional: to copy an id to the clipboard, select the id and do one of the following: press ctrl+c. right-click and select copy. optional: to download the execution artifacts for an executed process, such as process and document logs, click the download execution artifacts link. to return to the view from which you opened the dialog, click close.
