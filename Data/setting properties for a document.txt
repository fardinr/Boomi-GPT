you can use the set properties shape to set one or more properties for the current document. about this taskthe set properties shape is divided into two columns with actions to create a property and then assign one or more parameters to the property. you can add multiple parameter values to one property in order to concatenate a more complex value with dynamic content. follow the instructions to configure a set properties shape with property parameters.for information about each property and parameter the set properties shape supports, see the following topics: parameter value types document properties dynamic document properties mime properties process properties dynamic process properties procedure drag the set properties shape onto the process canvas. optional: in the set properties window, enter a custom display name. below properties to set, click the add property icon. the choose property dialog opens. select a property type, navigate to and select the desired property and click ok. select the newly added property on the left and then click the add parameter icon on the right (below property value). in the parameter value dialog, select a parameter type then configure the parameter and click ok. add additional properties and parameters as needed. optional: select a property or parameter and click the edit property or delete property icon to modify or remove it. optional: select a property and click the move parameter up or move parameter down icon to reorder it in the list. click ok.
