you can view linked documents in process reporting. procedure select manage > process reporting select executions from the list on the left. change the search options and filters to locate the process execution that you need. in the results table, find the process execution and click its link in the time column. navigate to one of the following locations: connection documents list documents with errors list click the actions icon and select view linked document. the document opens in the view linked documents dialog. select a connection from the show data for connector list. a list of documents appears. optional: for any document, click the actions icon and select view document. the document opens in the document viewer dialog. click cancel.
