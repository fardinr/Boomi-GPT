use the uninstall wizard to delete atoms, molecules, or cloud molecules from windows. procedure do one of the following: go to start > all programs > boomi platform > <atom_name> > uninstall atom. go to start > all programs > boomi platform > <molecule_name> > molecule uninstaller. go to start > all programs > boomi platform > <cloud_molecule_name> > cloud uninstaller. navigate to the directory in which you installed the atom, molecule or cloud molecule and run uninstall.exe. if a windows alert dialog opens asking you to confirm that you want to run the file, click yes. the uninstall wizard appears. click next then click finish.
