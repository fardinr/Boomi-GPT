you can add notes, each containing up to 250 characters, to a process for documentation purposes. procedure to add a note, click the add note link on the process canvas. a note box opens in the canvas.you can perform the next two steps in either order. type the note text in the box. move the note to the desired location on the canvas: pause on the note icon . drag the note icon and drop it at the desired location. click save. the note is added. to view the contents of the note, pause on the note icon. to move the note, follow the instructions in step 3. resultsnote: you can edit or delete a note only if you are its author or an administrator. to edit a note, pause on the note icon and click edit note. to delete a note, pause on the note icon and click delete note. then, in the confirmation dialog, click ok.
