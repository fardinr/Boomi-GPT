create map components for reference in map shapes where needed for converting data from one layout or format to another. procedure on the build page, click new. the create component dialog opens. select map from the type list. type a name for the map. the maximum length is 255 characters. optional: if you do not want the map to be stored in the default folder, click the browse button and select the desired folder. click create. the map appears on a tab with the name you entered. click save.
