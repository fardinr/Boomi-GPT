if there are changes to the structure of documents cached in an integration, you may need to edit indexes or keys in the document cache component. procedure open the document cache component. pause on the name of an index and click edit. edit the name of the index and click ok. if you delete the index name, it appears as “unnamed”. all previously saved blank index names also appear as “unnamed”. pause on the name of a key and click edit. do one or more of the following: edit the name of the key and then click ok. click the browse icon, select a different element, and then click ok. click save. repeat steps 2-6 as needed for other indexes and keys. save the document cache component.
