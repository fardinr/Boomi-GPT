cloud owners can stop an atom worker during its scheduled run time. procedure select manage > atom management. select an active atom cloud from the list on the left. in the runtime list, click atom workers. in the atom workers panel, locate the atom worker in the name column, click and select stop atom worker. a confirmation dialog opens. click ok. when the atom worker stops, it no longer appears in the active view and does appear in inactive. an entry is added to the cloud’s audit log.note: if you attempt to stop an atom worker that has been recently active because it appears to be “hanging,” then the atom worker is not stopped. if the atom worker has not been active recently, the atom worker is already terminated. the atom worker is removed from the active view and appears in the inactive view with an aborted status.
