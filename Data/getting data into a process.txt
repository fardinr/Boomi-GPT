on the left side of the canvas is the start shape. the start shape contains the inbound connector or other processing options that brings data into the process. a process can retrieve data from an application or a data source such as disk, mail, http, ftp, sftp, trading partner or database. a process can also be configured to listen for processing requests in real-time. procedure open a process. every process contains a start shape . click the start shape to configure the type of connector or other option that gets data for this process. review the connectors topic for more specific information about a particular type of connector. review the start shape topic for information about other options for getting data. click save.
