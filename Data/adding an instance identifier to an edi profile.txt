use the data elements tab to add instance identifiers (by occurrence and/or by qualifier) to a loop or segment in an edi profile.procedure open the edi profile. on the data elements tab, expand the tree and select the loop or segment. click the blue arrow icon and select add instance identifier. the add/edit instance identifier dialog opens. optional: to identify by an occurrence, select identify by occurrence and set occurrence by doing one of the following: click the first radio button and type or select a number. click the last radio button. optional: to identify by a qualifier: select identify by qualifier and click select qualifier to open the select qualifier dialog. optional: to filter the tree, type in the search field. expand the tree as needed. select a qualifier. click save. click save. after you have added the instance identifier you can do either of the following: edit the instance identifier. delete the instance identifier. click save to save the profile.
