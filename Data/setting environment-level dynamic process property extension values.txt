set extension values for dynamic process properties for an environment. before you beginyou must have the environment management privilege or environment management full access custom role to perform this task. you are unable to perform this task if your user is assigned with the environment management read access custom role. procedure select manage > atom management. select an environment from the list on the left. from the environment management screen, click environment extensions. optional: do one of the following: to filter the controls below to show only extensions defined for a particular process: click choose the desired component adjacent to process filter. select the process in the component chooser. to set extension values for a particular process in an installed, attached multi-install integration pack: in the process type list, select multi installation integration packs.note: this list option displays only if your multi-installation integration packs are installed on your account. click choose the desired component adjacent to process. select the process in the component chooser. select the dynamic process properties tab. enter a value for each property. when you are finished setting extension values, click ok.
