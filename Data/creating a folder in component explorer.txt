you can create hierarchical folders in component explorer to organize groups of components. about this taskyou can create new folders within existing folders only. for example, the boomi folder is primary. when you create a new folder, it appears on the next level as the secondary. also, you can create folders inside the secondary folders. procedure select the folder in which you want to create a folder and click the blue arrow next to it. select new folder. if a folder has write restrictions on it and if you are not assigned a user role that allows write access to the folder, this option is not available. the new folder dialog opens. type a name for the folder and click ok. the new folder inherits the folder permissions of its parent.
