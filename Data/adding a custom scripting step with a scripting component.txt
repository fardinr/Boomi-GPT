to add a custom scripting step with a scripting component to a data process shape, select the process type custom scripting and the script source use scripting component. procedure add a data process shape to your process or edit an existing data process shape. the data process shape dialog opens. click the add step icon. in the process type list, select custom scripting. for script source, select use scripting component. in the process script field, do one of the following: to locate an existing process scripting component, click in the field. to create a new process scripting component, click create. a new process scripting component opens in a separate tab. when you are finished, click ok to save the data process shape.
