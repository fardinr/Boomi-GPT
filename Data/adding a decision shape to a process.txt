add a decision shape to a process to route documents based on a true/false comparison of two values. procedure drag the decision shape onto the process canvas. the decision properties dialog opens. optional: enter a custom label. click the browse icon to the right of the first value field. the parameter values dialog opens. select the type of value. depending on the type of parameter that you select, you may have to enter more data. see the parameter values topic for more information. select a comparison operator. click the browse icon to the right of the second value field. the parameter values dialog opens. select the type of value. depending on the type of parameter that you select, you may have to enter more data. click ok. in the process, connect the true and false paths to the appropriate next shapes.
