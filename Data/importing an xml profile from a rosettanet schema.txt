the steps for importing an xml profile vary depending on the source file or schema. about this taskto import from a rosettanet schema: procedure on the build page, click new. on the create component screen, name the component and select profile as the type and xml as the profile format. on the tab named for your xml profile, click the import button. in the build xml profile from field, select rosettanet, and click next. select the rosettanet message type that you want to import. messages are categorized by message type and then version number. the version numbers in the xml import wizard match rosettanet version names. click finish. click save. next steps if you need to re-import a profile to incorporate updates, the updated profile might affect the connections in any map where it is used.
