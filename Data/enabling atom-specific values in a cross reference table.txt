setting cross reference table extensions is the preferred method to the steps shown here for replacing a set of values in a cross reference table. procedure create or open a cross reference table component. click enable atom-specific values. after being enabled, this option cannot be turned off. click ok. an atom name column is added to the table. a list box and two additional icons appear at the top of the table. select the check boxes for the rows that will use values from the atom. click the apply atom to selected rows icon. the atom's name appears in the first column. to remove an atom from a row, select its check box and click the reset selected rows to default icon. click save.
