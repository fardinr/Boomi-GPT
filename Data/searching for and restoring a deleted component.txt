you can search the component explorer for components that have been deleted, then restore the deleted component that you need. procedure on the build page in the component explorer, click the filter button. under the deleted components filter category, turn on the show deleted components filter option. all categories and folders that contain deleted components, as well as the deleted components themselves, appear in the component explorer. note that active components are also included in filter results. locate the component that you want to restore by clicking the plus sign icon next to the folder or category in which it is located. once you find the component that you are looking for, click the arrow next to it to expand the options menu. if it is a deleted component, there is a restore option available on the menu.if the folder in which the component is stored has write restrictions on it and if you are not assigned a user role that allows write access to that folder, this option is not available. do one of the following: click restore and click ok to confirm. double-click the component to open it in the process canvas. click save or save and close. the component is restored to its original location. repeat steps 3â€“5 for each component that you want to restore. in the component explorer, click filterand then click clear all filters.
