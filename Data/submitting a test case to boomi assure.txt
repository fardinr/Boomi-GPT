to submit a test case to boomi assure, run a process in test mode then submit it. about this taskboomi assure must already be enabled in the account. you must have the boomi assure privilege, which is part of the standard user role, enabled to submit a test to boomi assure. procedure open a process and run it in test mode. after the test finishes, inspect the results. click submit test case. a confirmation dialog opens. review the reminder. optional: if you do not want to see the reminder again, turn on the don't ask again? check box. click send data. the test is submitted to boomi assure. optional: if you change your mind, click cancel in the information dialog. optional: to return to edit mode, click the back button next to the process name.
