add published components to an integration pack on the deploy > integration packs page to make them shareable to an environment for use by managed accounts. about this taskcomponents must first be packaged from the packaged components page in order to be added as integration packs, and must be marked shareable when the packaged was created. procedure in the integration packs page, select the integration pack that you want to add packaged components to. under the packaged components heading, click the add packaged component button. the add packages to integration pack dialog opens. optional: to filter the list of packaged components, type in the filter components field. as you type, the list is filtered to include only components whose names partially or completely match the typed characters. to discontinue filtering and restore the unfiltered list, click . select the check box for each packaged component that you want to add to the selected integration pack. you can add a maximum of 100 packaged components. the most recent shareable version of each packaged component is added to the integration pack. click add n selected packages. the dialog closes, and the newly added packages appear in the packaged components list.
