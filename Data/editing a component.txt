edit components in the build page. procedure in the component explorer, navigate to the folder containing the component. double-click the component. if the component is in a folder without write restrictions on it, or if you are assigned a user role that allows write access to the folder, the component opens on a tab on the process canvas and is editable. if the component is in a folder with write restrictions on it and if you are not assigned a user role that allows write access to the folder, the component opens in read-only mode. to edit the component's name, click in the text box next to the component type and type a new name. the maximum length is 255 characters. to edit the component's description, click edit description here and type a new description. click save or save and close. for information on how to configure different types of components, see the specific topics in the build section of the table of contents.
