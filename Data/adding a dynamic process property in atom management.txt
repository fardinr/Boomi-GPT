use the deployed processes panel to add a dynamic process property. before you begin you must have the atom management privilege to perform this task. if you have the atom management read access privilege, you cannot perform the task.you can add dynamic process properties in this dialog. you cannot add process property components. procedure go to manage > atom management. select the atom, molecule, or cloud from the list on the left. on the deployed processes panel, select the process and click the blue arrow next to it. select edit process properties. the edit process properties dialog opens. click new property. an empty row appears above the new property button. type the name of a dynamic process property in the property name column. optional: type a value for the property in the property value column. repeat steps 5-7 to add more dynamic process properties. click save.
